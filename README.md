# kronos_test
Versão feita por Ian. Kronos é um app Flutter offline-first para rastrear hábitos de estudo e foco. Construído com Clean Architecture e Riverpod, ele salva suas sessões localmente via Sqflite e sincroniza relatórios na API do GitHub Gists usando Dio. O controle definitivo do seu tempo de aprendizado em uma arquitetura robusta.
